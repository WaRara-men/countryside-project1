# 🏯 安中・侍の足跡 (Annaka Samurai Footsteps)

群馬県安中市の中山間地域に住む高齢者の「健康維持」と「安否確認」を、**「侍の修行（散歩）」**という前向きな文脈で解決するデジタル・プラットフォームです。

「見守られる（監視される）」という受動的な立場から、「修行して領地を守る（歩く）」という能動的な立場へと高齢者の意識を転換し、同時に離れて暮らす家族に究極の安心を届けます。

---

## 🌟 主な機能

### 👤 高齢者用アプリ（修行モード）
- **出陣・帰還システム**: ボタンひとつで修行（散歩）を開始。開始地点を即座に家族へ共有。
- **侍ランク（ゲーミフィケーション）**: 累計歩数に応じて「新米門下生」から「伝説の将軍」へと階級がアップ。歩くモチベーションを維持します。
- **歩数ベースの目標設定**: 高齢者に馴染みのある「歩数」で目標（1,500歩 / 4,500歩 / 10,500歩）を選択可能。
- **帰り際ボイス診断**: 帰宅時の「ただいま」の声から、元気度をAIが判定。
- **オフライン対応**: 電波の悪い中山間地でも位置データを一時保存し、復帰時に自動同期。

### 🏠 家族用ダッシュボード（安否確認）
- **リアルタイム地図追跡**: 父さんの現在地をGoogle Maps上でリアルタイムに捕捉。地図は自動的に移動を追従します。
- **ユニバーサル住所表示**: OpenStreetMap APIを活用し、現在地の具体的な住所（市区町村・町名）を表示。
- **領内ステータス**: 「安中宿付近」「高崎方面（遠征中）」など、侍としての現在地を独自判定して表示。
- **健康バイタルグラフ**: 過去1週間の元気度（声のハリ）を日次で集計し、グラフ化。
