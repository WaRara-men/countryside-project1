# 🏯 安中・侍の足跡 開発ステータス (2026/03/19)

## 🎯 プロジェクトのゴール
群馬県安中市の高齢者が「侍の修行」として楽しく歩き、その安否と位置情報を家族がリアルタイムかつ正確に（地名レベルで）把握できるプラットフォームの完成。

## 📍 現在の地（進捗状況）

### ✅ 侍修行アプリ (Elderly App)
- **【真打】筆文字演出（侍フォント）**: `Yuji Syuku` 導入。「いざ出陣」「門下生」などの文字が力強い筆致に。
- **【守護】凶獣接近アラート**: 領内の危険エリア（赤い円）の200m圏内へ侵入した際、画面に「⚠️危険接近」と警告を発動。
- **【進捗】中山道・宿場町進捗**: 現在地を正確なkmで計測し、「安中宿〜坂本宿」の道のりを可視化。
- **【診断】ボイス元気度**: 帰宅時の第一声で元気度を診断し、家族へ自動報告。

### ✅ 家族見守りアプリ (Family App)
- **【可視化】領内の脅威マップ**: 凶獣（クマ・イノシシ）の最新出没エリアを赤い円で描画。足跡（現在地）との重なりを一目で把握。
- **【整理】修行絵巻アコーディオン**: 履歴が増えても地図が見やすいよう、最新3件以外を折りたたむスッキリUI。
- **【追跡】最新位置 Pan-To**: 侍（高齢者）の移動に合わせて、家族の地図が自動で中心を追従。
- **【精度】住所・地名併記**: 「番地レベルの住所」と「〇〇宿付近」の歴史的名称を同時に表示。

### ✅ 千里眼システム（自動情報同期）
- **【24H監視】AI自動データ化**: GitHub Actions が毎時、領内の動物ニュース（Gaccom等）を巡回。
- **【解析】Gemini AI 連携**: ニュース文章から住所を読み取り、瞬時に「緯度・経度・危険半径」へ変換し自動登録。
- **【復旧】依存関係の修正 (2026/03/20)**: `dotenv` 不足とインポートパス不備による Actions エラーを解消し、同期が正常稼働。
- **【強化】プッシュ・トリガー**: `main` ブランチへのプッシュ時に即座に同期が実行されるようワークフローを強化。
